Разработчики проекта Android-x86, в рамках которого силами независимого сообщества осуществляется портирование платформы Android для архитектуры x86, представили первую экспериментальную сборку на базе Android 4.0.1.

::cut::

Загрузочный iso-образ (191 Мб) на базе Android 4 оптимизирован для планшетов на базе платформы AMD Brazos, таких как MSI WindPad 110W. В настоящее время поддерживается Wifi, сенсорный экран, мультитач-навигация и использование OpenGL ES для аппаратного ускорения графики для карт AMD Radeon. Аппаратная акселерация графики для карт Intel пока не реализована. Также ещё нет поддержки звука, камеры и Ethernet.

Следует отметить, что компания Intel собственными силами ведёт работу по адаптации Android 4 для платформы x86. Основные усилия Intel нацелены на реализацию оптимизаций для систем на базе процессоров Atom (проект Android-x86 пока сосредоточен на платформе AMD Brazos). Из достижений Intel можно отметить реализацию в Native Development Kit (NDK) поддержки выполнения кода для архитектуры x86, включая оптимизации с задействованием инструкций MMX/SSE/SSE2/SSE3 и возможность сборки универсальных исполняемых файлов, которые можно запускать на системах x86 и ARM. Кроме того, проведена адаптация приложения Android Market, в эмулятор платформы Android добавлена поддержка прототипов устройств на базе архитектуры x86.

Подробный обзор новшеств Android 4 был опубликован в прошлом анонсе, из наиболее интересных улучшений можно выделить:

  • переработанный внешний вид;
  • новый интерфейс управления задачами с поддержкой принудительного завершения приложений;
  • переработанная область уведомлений;
  • программный интерфейс для распознавания лиц;
  • Android Beam для быстрого обмена сообщениями по NFC;
  • режим Wi-Fi Direct для связи двух устройств;
  • наглядная система контроля расхода трафика;
  • значительно переработанное приложение для работы с камерой;
  • расширение возможностей и производительности web-браузера;
  • унифицированный набор инструментов для формирования интерфейса пользователя телефонов, планшетов и других устройств, единый набор компонентов интерфейса и стилей.
