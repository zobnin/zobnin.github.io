Ram Manager - это оптимизатор работы с оперативной памятью. Он изменяет низкоуровневые настройки Android таким образом, чтобы установить правильный баланс между скоростью запуска новых приложений и производительностью уже работающих. Программа предлагает несколько профилей управления памятью, которые вы можете использовать в различных ситуациях:

  • Balance - установка оптимальных настроек управления памятью, которые позволят сделать работу устройства более быстрой и отзывчивой. Профиль рекомендуется для каждодневного использования.
  • Balance (with more free memory) - профиль, аналогичный предыдущему, но предназначенный для устройств с объемом оперативной памяти меньше 512 Мб. Выбор этого профиля приводит к установке настроек, благодаря которым в системе появится больше свободной памяти (а значит запуск новых приложений будет происходить быстрее), но это негативно скажется на скорости переключения между уже запущенными приложениями (так как система будет раньше выгружать их из памяти).
  • Balance (with more multitasking) - профиль, активация которого приводит к обратному результату. Уже запущенные приложения дольше остаются в памяти, а значит переключение между ними происходит быстрее, но для запуска новых приложений требуется больше времени из-за отсутствия достаточного количества свободной памяти. Рекомендуется для устройств с объемом памяти больше 512 Мб.
  • Hard Gaming - профиль, предназначенный для запуска тяжелых игр (таких как GTA3). За счет выгрузки фоновых приложений в системе освобождается большое количество памяти, которое будет доступно игре.
  • Hard Multitasking - профиль, предназначенный для одновременной работы со многими приложениями. Выгрузка запущенных приложений из памяти происходит только в крайнем случае.
  • Default (Samsung) - стандартные настройки, используемые в смартфонах Samsung. Также подходят для других устройств.
  • Default (Nexus S) - стандартные настройки, используемые в смартфоне Google Nexus S.
  • Default - стандартные настройки для текущего смарфтона.

Авторы приложения заявляют, что программа может сделать устройство более отзывчивым, а запуск приложения более быстрым, так же упоминая о том, что благодаря профилю Hard Gaming можно добиться очень плавной работы GTA3 даже на Samsung Galaxy S и смарфтонах схожих конфигураций. Бесплатная версия приложения предлагает выбор только опции Default, для получения доступа ко всем остальным придется купить Ram Manager Pro за 60 рублей.
