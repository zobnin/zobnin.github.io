Долгое время в Android вообще не было штатного способа сделать скриншот экрана, что привело к рождению множества насмешек, особенно со стороны приверженцев iOS. С появлением Android 4.0 Ice Cream Sandwich сделать скриншот стало возможным одновременно нажав клавиши включения и уменьшения громкости, благодаря чему проблема решилась. Но как быть пользователям более ранних версий Android? В этой статье мы расскажем о пяти различных способах сделать скриншот и вы сможете выбрать тот, который подходит вам больше.

Приложение

Наверное самый простой способ это воспользоваться одним из многих приложений для снятия скриншота из маркета. Сегодня они уже не требуют прав root и способны работать на любом смартфоне. Из примеров можно привести платную No Root Screenshot It и бесплатную Screen Capture Shortcut Free.

Если ваш смартфон или планшет рутован, вы сможете выбрать из десятков подобных приложений поискав в маркете по ключевому слову "screenshot" или "screen capture".

AirDroid

Второй способ для тех, кто хочет получить скриншот прямо на компьютер используя сеть WiFi. Для этого просто установите приложение AirDroid, запустите, откройте в браузере показанный на экране адрес и введите показанный пароль. Вы должны увидеть нечто вроде рабочего стола как показано на скриншоте выше, кликайте по иконке "Скриншот".

Терминал (Android 4.X)

В Android 4.X скриншот можно сделать и прямо из окна терминала. Для этого запустити Terminal Emulator (если он установлен) и наберите следующую команду: 

$ creencap -p /sdcard/screenshot.png
Скриншот будет сохранен в файл screenshot.png на карте памяти.

Android SDK

Если у вас установлен Android SDK вы можете сделать скриншот в любой версии Android с помощью инструмента DDMS. Для этого подключите смартфон к компьютеру с помощью USB-кабеля, включите режим отладки (Настройки - Для разработчиков - Отладка по USB). Далее откройте каталог, в который вы установили Android SDK, в нем откройте каталог tools и запустите файл ddms. В открывшемся окне выберите в меню "Device - Screen Capture".

ADB (для пользователей Linux)

Если вы используете Linux, то Android SDK и DDMS вам не понадобятся, достаточно иметь adb и ffmpeg. Подключите смартфон с помощью USB, а затем наберите следующие кманды: 

$ adb pull /dev/graphics/fb0
$ ffmpeg -vframes 1 -f rawvideo -pix_fmt rgb32 -s 720x1280 -i fb0 screenshot.png
Вместо 720x1280 укажите разрешение своего смартфона или планшета. Скриншот будет сохранен в файл screenshot.png.
